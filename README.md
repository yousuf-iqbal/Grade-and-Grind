# Grade & Grind

## Description
A web based freelancing platform for university students, connecting them with clients for short term gigs, skill based work, and real time communication.

## Tech Stack
- **Backend:** Node.js
- **Frontend:** React vite
- **Database:** MSSQL

## How to Run

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm run dev
